# 纯粹记账（Vue）



## 描述

这是一个真实的记账项目，用于学习和记录

技术：vue2+vue-router+vux+typescript

## 使用

### 开发

只需要执行以下命令就可以在 http://localhost:3333 中看到

```bash
yarn dev
```

### 构建

构建该应用只需要执行以下命令

```bash
yarn build
```

然后你会看到用于发布的 `dist` 文件夹被生成。

### 部署到 Netlify

前往 [Netlify](https://app.netlify.com/start) 并选择你的仓库, 一路 `OK` 下去，稍等一下后，你的应用将被创建.
